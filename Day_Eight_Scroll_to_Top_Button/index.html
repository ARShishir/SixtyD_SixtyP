ðŸ”¹ How It Works

When the user scrolls down more than 300 pixels, the button appears at the bottom-right corner. On click, a JavaScript function triggers a smooth animation back to the top of the page.

ðŸ”¹ Example Code Snippet


window.addEventListener("scroll", () => {
  if (window.scrollY > 300) {
    scrollBtn.classList.add("show");
  } else {
    scrollBtn.classList.remove("show");
  }
});

scrollBtn.addEventListener("click", () => {
  window.scrollTo({ top: 0, behavior: "smooth" });
});
